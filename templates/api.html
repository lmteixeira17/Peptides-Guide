Endpoint

A API pública v1 retorna o catálogo completo em JSON quando solicitada por ferramentas, scripts ou chamadas com Accept: application/json.

{{ peptide_count }} peptídeos individuais
{{ stack_count }} combinações recomendadas
{{ reference_count }} referências científicas

Abrir JSON bruto

Uso rápido

fetch("{{ api_endpoint_url }}", {
  headers: { "Accept": "application/json" }
}).then(response => response.json())

O endpoint legado {{ api_legacy_url }} continua ativo por compatibilidade.

Estrutura

  • version: versão do formato de dados.
  • source: URL canônica do guia.
  • peptides: lista de peptídeos com categoria, status, mecanismo, benefícios, efeitos, dosagens e referências.
  • stacks: combinações com objetivo, nível, duração, aplicação, sinergia, peptídeos e referências.
Uso educacional: os dados são uma referência científica informativa e não substituem avaliação médica profissional.
